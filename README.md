Zokugun Theme for Visual Studio Code
====================================

## Supported Languages

- Bash
- CSS
- Go
- HTML
- JavaScript
- JSON
- Kaoscript
- Makefile
- Markdown
- nginx
- PHP
- Pug
- Python
- Sass
- TypeScript
- YAML
